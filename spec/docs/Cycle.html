cycle_duration_us :: Stream Word32 #

Length of a breath cycle in microseconds. Derived from the BPM user input stream with limits.

inhale_duration_us :: Stream Word32 #

Length of an inhale in microseconds. Derived from the BPM and I:E ratio.

exhale_duration_us :: Stream Word32 #

Exhale duration. Sum of this and inhale_duration_us is cycle_duration_us.

Could also use a similar computation to inhale_duration_us, but we'd rather have any loss in the integral division of inhale_duration_us be compensated for here. i.e. we always have

exhale_duration_us + inhale_duration_us = cycle_duration_us
